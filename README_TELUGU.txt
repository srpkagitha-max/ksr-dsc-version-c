KSR DSC Complete Version C Project

GitHub upload:
1. ZIP extract చేయండి.
2. GitHub repositoryలో Add file → Upload files.
3. ZIP కాదు. Extract అయిన files అన్నీ upload చేయండి:
   index.html, admin.html, style.css, firebase-config.js, admin.js, student.js, firestore-rules-final.txt, README_TELUGU.txt
4. Commit changes నొక్కండి.

Netlify:
1. Netlify → Add new site → Import from GitHub.
2. Repository select చేయండి.
3. Build command blank.
4. Publish directory blank/root.
5. Deploy.

Important:
Adminలో exam మళ్లీ Upload Exam + Generate Codes చేయాలి.
Generated Codes box లో వచ్చిన NEW codes మాత్రమే studentsకి పంపాలి.
